## Run Locally

Download

- node
- npm
- typescript / tsc

Clone the project

```bash
git clone https://github.com/Kkwagh-Coders/interview-experience-api.git
```

Change to Project Directory

```bash
  cd interview-experience-api
```

Install Dependencies

```bash
  npm install
```

Start React App

```bash
  npm run start:dev
```
